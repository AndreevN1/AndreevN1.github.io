Перечень сокращений и обозначений

В настоящей пояснительной записке применяются следующие сокращения и обозначения:

ЛВС— локальная вычислительная сеть
СКС— структурированная кабельная система
ТКШ— телекоммуникационный шкаф
ПЗ— пояснительная записка
ДП— дипломный проект
ПК— персональный компьютер
UTP— Unshielded Twisted Pair (неэкранированная витая пара)
PoE— Power over Ethernet (питание по витой паре)
VLAN— Virtual Local Area Network (виртуальная локальная сеть)
STP— Spanning Tree Protocol (протокол связующего дерева)
LACP— Link Aggregation Control Protocol
SVI— Switched Virtual Interface
DMZ— Demilitarized Zone
NAT— Network Address Translation
DHCP— Dynamic Host Configuration Protocol
DNS— Domain Name System
NTP— Network Time Protocol
HTTP— Hypertext Transfer Protocol
HTTPS— Hypertext Transfer Protocol Secure
FTP— File Transfer Protocol
ISP— Internet Service Provider
CCTV— Closed-Circuit Television (видеонаблюдение)
MGMT— Management (управление)
HTML— HyperText Markup Language
CSS— Cascading Style Sheets
