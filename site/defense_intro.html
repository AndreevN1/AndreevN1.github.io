О дипломном проекте

  • Дипломный проект по специальности 09.02.06 «Сетевое и системное администрирование»
  • Студент: Андреев Никита Александрович, группа СА-22
  • Руководитель: Лиштван А.А.
  • Тема: Проектирование СКС для офиса интернет-магазина, с сегментированием сети

Цель и задачи проекта

  • Цель: разработать СКС для офиса интернет-магазина с VLAN-сегментированием
  • Спроектировать сеть для двух независимых объектов B1 и B2
  • Составить кабельный журнал на 60 линий с маркировкой
  • Разработать схемы L1, L2, L3 и фасад ТКШ
  • Настроить VLAN, trunk, STP, UserGate F100
  • Настроить сервисы: DHCP, DNS, NTP, HTTP, FTP, Email
  • Рассчитать стоимость: материалы + монтаж + оборудование
  • Создать сайт ПЗ на HTML5/CSS3, опубликовать на GitHub Pages

Актуальность

  • Надёжная сетевая инфраструктура — ключевое конкурентное преимущество интернет-торговли
  • VLAN-сегментирование обеспечивает безопасность и управляемость без лишнего оборудования
  • UserGate F100 — сертифицированный ФСТЭК российский МЭ, требование импортозамещения
  • Публичные серверы (HTTP, FTP, Email) изолированы в DMZ для защиты внутренней сети

Объект и предмет исследования

  • Объект: корпоративная сеть офиса интернет-магазина на двух объектах B1 и B2
  • Предмет: методы проектирования, настройки и документирования СКС с VLAN
  • Методы: анализ технической документации, моделирование в Cisco Packet Tracer
