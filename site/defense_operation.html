Конфигурация коммутатора SWM1

  • SVI-интерфейсы для каждого VLAN — межвлановая маршрутизация
  • Gi0/1 → SW1 (trunk: VLAN 100,200,999), Gi0/2 → SW2 (trunk: VLAN 100,300,999)
  • SWM1 — корневой мост STP (приоритет 4096)

UserGate F100 — зоны безопасности

  • inside (уровень 100): 10.255.1.1/30 → SWM1
  • outside (уровень 0): 172.16.1.1/28 → R1 (интернет)
  • DMZ (уровень 50): 192.168.10.1/28 — публичные серверы
  • NAT Masquerade: inside/DMZ → outside; ACL: запрет internet → inside

Сетевые сервисы (Cisco Packet Tracer)

  • DHCP — пул B1_USERS: 192.168.40.10-30, шлюз 192.168.40.1
  • DNS — зона corp.local: A-записи для всех серверов
  • NTP — синхронизация времени оборудования
  • HTTP/HTTPS — корпоративный сайт в DMZ
  • FTP — файловый обмен в DMZ (cisco/ftpuser)
  • Email (SMTP/POP3) — домен corp.local
