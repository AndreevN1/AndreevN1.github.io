Стек технологий

  • HTML5 — семантическая разметка, 16 страниц
  • CSS3 — Flexbox-вёрстка, CSS-переменные, медиазапросы
  • Без JavaScript — универсальная совместимость, быстрая загрузка
  • GitHub Pages — бесплатный хостинг, CI/CD через git push

Цветовая схема (colorscheme.ru)

  • Тип: монохромная (моно), основной тон — бирюзово-зелёный
  • #2F9292 — основной, #07444A — тёмный, #83E3E3 — светлый акцент
  • Предпросмотр светлой и тёмной тем подтверждает читаемость
  • CSS-переменные :root позволяют менять палитру в одном месте

Компоновка (Flexbox)

  • Шапка: position:fixed, 92px, логотип + название + тема
  • Левое меню: 270px, тёмный фон, 42 пункта, overflow:auto
  • Контент: flex:1, justify-content, overflow-y:auto
  • Правое меню: 200px, шпаргалки для защиты ДП
  • Адаптивность: медиазапросы для 1200/768px
